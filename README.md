# course-enrollment-manager

Course Enrollment Manager is a backend focused webapp that allows students to enroll/disenroll in courses, manage their profile, and view grades. Instructors may
see which courses they are teaching and view class rosters, manage their profile, and give students grades. The webapp is hosted on an AWS LAMP EC2 instance
utilizing AWS RDS to store the database.
